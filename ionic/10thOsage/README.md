
# Setup Process

1. Install node.js
2. Run the following commands (osx)

	$ npm install -g cordova ionic
    $ ionic hooks add
    $ ionic platform add ios
	$ npm install -g ios-sim

3. To create the project, run:

    $ ionic start 10thOsage blank

4. To build and run:

    $ ionic build ios
    $ ionic emulate ios
